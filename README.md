# kojaping 
🛏편리한 숙소 예약을 위한 숙소예약 사이트 프로젝트입니다.🏡  


## 📌프로젝트
### 소개
- 숙소를 올리고 관리할 수 있어요.
- 엘라스틱서치를 통해 강화된 검색엔진을 통해 쉽고 빠르게 숙소를 골라보아요.

### 기능 소개
회원가입/로그인  
- 회원가입시 게스트회원과 호스트회원을 구분하여 하나의 디비에 저장합니다.
- 로그인시 회원이 구별되어 페이지로 이동합니다.
  
마이페이지  
- 정보수정, 회원퇄퇴가 가능하며 회원탈퇴시 논리삭제로 구현하였습니다.
    
게시판
- 내가 올린 숙소를 관리할 수 있어어요.
